Basic User Commands

register

Syntax: register <name>

Example: register voicex

unregister

Syntax: unregister <name>

Example: unregister voicex

post

Syntax: post <msg>

Example (simple post): post hello world

Example (tagged post): post voicex is free. #voicex

Example (anonymous post): post voicex is free. #annon

view

Syntax: view <post-id>

Example: view 3

delete

Syntax: delete <post-id>

Example: delete 3

reply

Syntax: reply <post-id> <msg>

Example: reply 3 this is my reply.

Example (public reply): post this is my public reply. #public

Example (anonymous reply): post this is my public and anonymous reply. #public #annon

Example (anonymous reply): post this is my public and anonymous and tagged reply about voicex. #public #annon #voicex

search

Syntax: search <keywords>

Example: search kenya elections

follow

Syntax: follow <@name/#tag>

Example (following an organization): follow @stanford

Example (following a tag): follow #employment

unfollow

Syntax: unfollow <@name/#tag>

Example (unfollowing an organization): unfollow @stanford

Example (unfollowing a tag): unfollow #health
